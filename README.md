# 🌊 陆地-潮间带-浅水区一体化两栖采样船
# Land-Intertidal-Shallow Water Integrated Amphibious Sampling Robot

![Status](https://img.shields.io/badge/Status-Under%20Development-orange)
![Hardware](https://img.shields.io/badge/Hardware-ESP32-blue)
![Mechanical](https://img.shields.io/badge/Mechanical-3D%20Printed%20%7C%20PETG--CF-green)

本项目致力于解决目前环境监测领域的最大盲区——**“潮间带与浅水水域过渡区”**的自动化采样难题。通过独特的轻量化设计、可升降履带悬挂以及低压强底盘，本机器人能够在涨��的浅水区与退潮的极软淤泥滩涂中自由穿梭，完成沉积物与水样的精准采集。

---

## 📍 场景痛点与研发初衷

**潮间带（涨潮被淹、退潮露出的区域）**是污染物入海的最后屏障和生态敏感区。但现状面临极大挑战：
*   **涨潮浅水（0.5-3m）**：常规吃水深的采样船极易搁浅。
*   **退潮滩涂**：无人机无法降落，人工“蹚泥”效率极低且陷脚危险，采样点难以精准定位。

**本方案**：设计一款全地形两栖采样小船，利用特殊的“滑板+升降履带”模式，结合高适配度的嵌入式系统，实现全地形的低成本、高可靠自动化采样。

---

## ⚡ 嵌入式与电控系统架构 (Embedded & Control System)

为了支撑复杂的升降底盘和多自由度采样，本项目的嵌入式系统围绕**高性价比、强扭矩输出与总线控制**进行设计。

### 1. 核心主控与通讯
*   **主控单元**：基于 **ESP32** 控制盒，提供充足的 GPIO 口、总线通信接口以及原生的 Wi-Fi/蓝牙 支持，便于后期接入远端遥控终端或地面站。
*   **视觉反馈**：4轴机械臂端头集成**微型摄像头**，配合 ESP32 的图传能力（如 ESP32-CAM 模块扩展），实现采样点的精准视觉定位。

### 2. 动力与底盘驱动系统
底盘采用“高转速电机 + 高减速比”的轻量化策略：
*   **驱动电机**：两台 **775 系列直流电机**（带编码器），配合 1:64 行星减速器（24V输出），经减速后输出转速约 80 RPM，兼顾水下推进与陆地越障扭矩。
*   **电机驱动器**：采用双 **IBT-2 高功率驱动模块 (43A)**，可承受履带在深陷淤泥时的瞬时堵转大电流，确保系统不烧毁。

### 3. 悬挂升降与姿态控制
四连杆悬挂系统需要极高的保位力矩与同步性：
*   **执行器**：左右各配备一台 **60kg·cm 级高压串行总线舵机**（如幻尔 STS 系列）。采用总线级联方式，大幅减少主控接线。
*   **闭环保护**：在最高和最低位设置**物理限位 + 霍尔传感器 (Hall Sensor)**反馈。当淤泥阻力过大导致过流时，电控系统可及时切断输出，保护机械结构。

### 4. 采样工作站控制
机械臂采用数字总线架构，4个自由度 (DOF) 完成“铲土”和“舀水”：
*   **大臂/腰部旋转**：采用 **60kg·cm** 总线舵机，提供稳定的举升力。
*   **小臂/手腕俯仰**：采用 **35kg·cm** 全防水总线舵机，确保即使侵入水中也能正常工作。
*   **末端快换执行器**：
    *   水样采集：继电器控制的 **PVC微型水泵**。
    *   泥土采集：**9g 微型舵机** 控制的 3D 打印小铲斗。
*   **样本收纳盘**：步进电机控制的旋转盘（类似左轮手枪弹巢），每次采样后自动步进一格，防止样本交叉污染。

### 5. 能源管理
*   **电池组**：**24V 8Ah 动力锂电池组**，放置于船型箱体最底部，极大降低整车重心，提高水上与滩涂行驶的抗倾覆能力。

## ⚙️ 核心机械创新点 (Mechanical Innovations)

### 1. 3D打印防陷履带 + EVA 浮力履刺
*   **原理**：摒弃传统重型履带。采用 25mm 节距的 **PETG-CF（碳纤增强 PETG）** 3D 打印链节。将**硬质 EVA 泡沫**（高密度泡沫砖）粘合或通过不锈钢螺丝锚固在履带背侧。
*   **水上模式**：EVA泡沫提供约 5-8kg 静浮力，同时在转动时充当“明轮/桨叶”，实现无螺旋桨的水中推进，防止海草缠绕。
*   **自动排泥**：链节上预留孔洞，且销轴（Φ4mm不锈钢空心管）间留有0.5-0.8mm间隙，驱动齿轮啮合时可自动将泥沙挤出，减少打印件磨耗。

### 2. “底盘触地”可升降悬挂系统
*   **结构**：由总线舵机驱动的**平行四边形四连杆机构**。
*   **滩涂工作模式**：进入淤泥区后，悬挂主动抬升履带，使船型底盘下降直接接触泥面。
    *   **极低压强**：整车底盘加上履带接触面积达 0.17 ㎡，25kg 满载状态下接地压强仅 **1.44 kPa**（远低于 2kPa 沉陷临界值），变身“带动力刷的雪橇”。
    *   **低重心采样**：底盘坐滩后，离地间隙降至 15mm，彻底消除水流晃动，确保机械臂精准作业。
*   **陆地越障模式**：悬挂降下，离地间隙升至 110mm，轻松跨越礁石与树根。

## 📐 整车物理参数 (25kg 方案)

| 参数项 | 规格 / 材质 |
| :--- | :--- |
| **车体尺寸** | 400mm x 340mm x 170mm |
| **整备重量** | 约 23.0 kg (满载设计 25kg) |
| **主车体材质** | 4mm壁厚 PETG-CF + 8mm碳纤加强管 (Gyroid填充) |
| **履带参数** | 节距 25mm, 宽度 110mm, 单侧接地长 ~200mm |
| **离地间隙** | 陆地模式 110mm / 淤泥底盘坐滩模式 15mm |
| **接地压强** | **~1.44 kPa** (防陷能力极强) |

### 质量分布评估
*   主仓体及碳纤骨架：~5.5 kg
*   履带系统 (含销轴)：~4.8 kg
*   动力与驱动组件：~2.2 kg
*   悬挂执行机构：~1.5 kg
*   4-DOF 机械臂：~3.2 kg
*   能源与电控箱：~3.5 kg
*   样本盘与作业裕量：~2.3 kg
